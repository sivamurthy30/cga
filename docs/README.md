# DEVA - Career Development Platform

## Overview
DEVA is an AI-powered career guidance platform that helps users navigate their professional development journey through personalized roadmaps, skill assessments, and premium features.

## Quick Start
1. **Backend**: `cd backend && bash run.sh`
2. **Frontend**: `npm start`
3. **Access**: http://localhost:3000

## Architecture
- **Frontend**: React.js with modern hooks and state management
- **Backend**: FastAPI with SQLAlchemy and ML integration
- **Database**: SQLite with planned PostgreSQL migration
- **ML**: LinUCB bandit algorithms for personalized recommendations

## Features
- 🗺️ Interactive career roadmaps
- 🎯 Skill assessments and quizzes
- 🤖 AI-powered recommendations
- 💼 Resume builder and interview prep
- 📊 Salary insights and market data
- 🎓 Advanced learning concepts
