## ⚡ Quick Start (5 Minutes)

### 1. Start Backend
```bash
python3 start_backend.py
```
Backend runs on: http://localhost:5001

### 2. Start Frontend
```bash

```
Frontend opens at: http://localhost:3000

### 3. Test LinUCB
```bash
python3 test_linucb.py
```
