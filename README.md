# DEVA - Career Development Platform

> AI-powered career guidance platform with personalized roadmaps and skill assessments

## 🚀 Quick Start

```bash
# Start backend
cd backend && bash run.sh

# Start frontend (in new terminal)
npm start

# Access application
open http://localhost:3000
```

## 📁 Project Structure

```
cga/
├── 📄 docs/           # Documentation
├── 🎨 src/            # React frontend
├── ⚙️ backend/        # FastAPI backend  
├── 📊 data/           # Shared data
└── 🎯 public/         # Static assets
```

## ✨ Features

- 🗺️ Interactive career roadmaps
- 🎯 Adaptive skill assessments  
- 🤖 AI-powered recommendations
- 💼 Resume builder & interview prep
- 📊 Salary insights & GitHub analytics
- 🎓 Advanced learning concepts

## 🛠️ Tech Stack

- **Frontend**: React, Zustand, GSAP, Framer Motion
- **Backend**: FastAPI, SQLAlchemy, Pydantic
- **Database**: SQLite → PostgreSQL
- **ML**: LinUCB algorithms, scikit-learn
